数据类型：

    数据类型决定：
        数据占内存字节数
        数据取值范围
        可以进行的操作

1. 基本数据类型 
     
    整型

        有符号整型
            有符号短整形 short
            有符号基本整型 int
            有符号长整型 long
        无符号类型
            无符号短整型
            无符号基本整型
            无符号长整型
    
    实型（浮点型）

        单精度型 float
        双精度型 double
    
    字符型 char
    
    枚举型 enum

2. 构造类型
    
    由基本类型按照一定规则组合而成的复杂数据类型，包括：

    数组型

    结构体型

    共用体类型

3. 指针类型

    更加灵活的数据类型，用来表示内存地址

4. 空类型 void

    表示不需要数据值，因而没有类型
